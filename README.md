# Installation

1. **Cloning a repository**:

   ```bash
   git clone https://github.com/Vlaek/DevSolutions.git
   cd DevSolutions
   ```

2. **Installing dependencies**:

   ```bash
   npm install
   ```

3. **Launching the application**:
   ```bash
   npm run dev
   ```
